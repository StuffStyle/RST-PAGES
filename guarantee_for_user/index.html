涡轮用户信息安全保证书

最后更新时间：2023年3月31日

一、声明

本保证书完全原创，其内容具有法律效应，任何个人或单位未经过同意前禁止使用于任何商用用途。

本保证书由回响项目组（以下称RS团队)编写，涡轮重铸科技(以下称涡轮)涡轮审核。

本保证书生效起始期为2023年4月2日，初版编写于2023年4月1日，其各版本保证书文件可在开源平台Github中查找源文件。

内部人员未经过任何允许，不得修改本页面任何源代码。如需修改，请申请修改权限！

二、使用

2.1 涡轮不会以任何形式的理由使用用户的任何隐私信息。

2.2 涡轮无权直接访问、增加、修改、删除、调用用户的隐私信息。

2.3 涡轮仅在用户使用一级密码、二级密码、服务器独立密钥授权后才可进行对用户隐私信息的相关操作。

2.4 涡轮仅能调用用户的开放信息（如用户上传的头像、名字、生日、服务器配置信息等)。

2.5 用户的开放信息在进行设置后才可公开，否则都默认隐藏。

2.6 涡轮不会使用用户的开放信息进行广告分析、第三方平台分享、大数据研究、AI模型训练等。

2.7 您的信息安全就是我们最重要的工作，我们会全力保障您的个人信息安全。

三、存储

3.1 中国大陆用户的个人信息统一存储在腾讯云广东区域（包括香港、澳门)服务器。

3.2 海外用户的个人信息统一存储在美国洛杉矶谷歌数据中心(不包括自主IP地区屏蔽)。

3.3 涡轮会将用户信息三重加密化，保证信息不泄露。

3.4 涡轮使用更好的服务提供商，以保障信息安全。

3.5 在必要时使用CDN等分发服务时，用户的信息处于加密阶段。

3.6 用户自行选择非涡轮产品的本地存储时，已脱离保证书保护信息的范围，涡轮不会保证信息安全。

3.7 本地信息存储在涡轮产品时，涡轮将全力保护你的个人信息。

四、不可抗力

4.1 中国大陆内协助警方、军方行使刑事权力时，涡轮将会把您的个人隐私信息全部移交至警方和军方。

4.2 海外内协助中央情报局、警方、调查局等行使相关权力时，涡轮会先询问您的意见，再把您的个人隐私信息全部移交。

4.3 遇到自然灾害、疫情等不可抗因素时，可能会造成的泄露事件将经过研究，再进行判决。

4.4 源服务器损坏导致的信息丢失，由服务提供商承担相关责任。
